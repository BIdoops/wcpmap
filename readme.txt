## WCPetroMap -PetroMap integration for WC MarketPlace.

Installation:

1) Download the plugin zip file ( You can download this repository as ZIP).
2) Install the plugin from disk in your wordpress installation ( Requires a previously an active WC MarketPlace installation).
3) Activated the plugin.

Activate WCPetroMap to your vendors.
1) Go to WCMp -> Settings -> Vendor -> Vendor Dashborad menu.
2) Check Activar PetroMap option and save *
3) Go to wordpress Widgets menu and set the "WCPMap: Ubicaci√≥n de la tienda del vendedor" in your lateral bar, to display the vendor store location
in his store page.

*Note: This will override Google Maps functionality. If you want to use Google Map later you'll need to uncheck this option.
